# hubot-gif

A hubot script that does the things

See [`src/gif.coffee`](src/gif.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-gif --save`

Then add **hubot-gif** to your `external-scripts.json`:

```json
[
  "hubot-gif"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
